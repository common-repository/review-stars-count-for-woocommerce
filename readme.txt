This plugin allows your visitors to see a visual overview of total stars/reviews given to a specific product.

== Description ==

Review Stars Count For WooCommerce enables your customers to see a visual overview of all the rating stars given to specific products. The plugin also allows them to mark any reviews on the product page as helpful or not by choosing thumbs up or thumbs down.

== Installation ==

1. Upload the "woo-star-count.zip" file to your "/wp-content/plugins/" directory.
2. Once uploaded, go to your WordPress admin and navigate to the plugins menu.
3. Once you are on the plugins screen, you will see Review Stars Count For WooCommerce listed there.
4. Just activate and start using the plugin configurations.

== Frequently Asked Questions ==

= Why should I use this plugin? =
We all like to know more insights about the products before we actually purchase them. This plugin helps your customers to know the exact number of ratings/reviews given to a specific product, allowing them to make informed purchasing decisions.
